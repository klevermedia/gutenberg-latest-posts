Lee&#39;s Recent Posts List

== Description ==

This plugin creates a custom Gutenberg block that allows users to search for and select recent posts. An inner block is then created with a "Read More" link.

Additionally, the plugin includes a WP-CLI command, enabling you to search for posts that contain the custom Gutenberg block.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
